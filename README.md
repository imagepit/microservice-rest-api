# マイクロサービスアーキテクチャ入門研修 REST APIサンプル

## 環境

項目|バージョン
---|---
JDK|OpenJDK 14
Spring Boot|2.3.4
Docker Desktop|19.03.12

## 使い方

**ホストOSでの実行**

```
https://github.com/imagepit/microservice-rest-api.git
cd microservice-rest-api
./gradlew bootRun
```

**コンテナでの実行**

```
https://github.com/imagepit/microservice-rest-api.git
cd microservice-rest-api
docker-compose up -d
```
